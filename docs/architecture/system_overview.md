# System Overview

この文書は、システム全体を短く把握するための説明です。実装の入り口は [../repository_guide.md](../repository_guide.md) にまとめています。

本研究では、聞き手の身体動作である頷きと首振りを IMU で検知し、直前の会話の流れに合う相槌音声を選んで、話し手に提示するシステムを提案する。システムは入力、処理、出力の 3 段階で構成される。

入力では、装着型デバイスに搭載した IMU から得た信号を用いて、肯定的な頷きだけでなく、否定や疑問を示す首振りも検知対象とする。映像は取得しないため、映像送信が前提の方式に比べて、見た目の情報を送らずに反応を伝えられる。

処理では、検知された動作と、直近の話し手発話の文字起こしに基づいて、相槌の候補音声から 1 つ、または返さないという判断を行う。これにより、単調な合図音で起きやすい文脈との不一致を減らすことを狙う。文字起こしは話し手音声を用いるため、運用条件によっては音声データの扱いに配慮が必要である。

出力では、選ばれた相槌音声をネットワーク経由で話し手端末に送り、聴覚的に提示する。画面表示に依存しないため、話し手は資料を見ながらでも会話のリズムを保ちやすい。
